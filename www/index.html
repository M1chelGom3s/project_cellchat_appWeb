Development Approach

This web application is still in beta version. Based on cellchat takes as input the expression data of cell populations are annotated and created from Seurat (V3). Thus cellchat combines the interaction of the ligand receptor and its cofactors to simulate intercellular communication. The analysis of the Cellchat model is adapted from the code of Suoqin Jin. (Inference and analysis of cell-cell communication using CellChat)

We will therefore use CellChat an R package designed to study intercellular communications from scRNA-seq data, CellChat provides features for data exploration, analysis and visualization. The 'CellChat' package has been compiled under R version 4.1.3 (2022-03-10).
